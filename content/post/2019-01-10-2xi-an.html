xian <- read_xlsx("E:/1R/website/Excel/xian.xlsx")
xian$property <- as.factor(xian$property)

getcolor <- function(xian) {
  sapply(xian$property, function(ppt){
    if (ppt == '景点') {
      "green"
    } else if(ppt == '车站') {
      "red"
    } else{
      "blue"
    }
  })
}
geticon <- function(xian){
  sapply(xian$property, function(ppt){
    if (ppt == '景点') {
      "eye"
    } else if(ppt == '车站') {
      "playstation"
    } else{
      "beer"
    }
  })
}
icons <- awesomeIcons(
  icon = geticon(xian),
  iconColor = 'white',
  library = 'ion',
  markerColor = getcolor(xian)
)

leaflet(data = xian) %>%

第一日

西安给人的总体印象就是什么都是一致的。包括以下专有名词：牛羊肉泡馍，biangbiang面，肉夹馍，棉花糖……

室友带我涨知识之 肉夹馍

由于学校工学部菜市场有个盗版的西安肉夹馍，室友提醒我出发前尝尝并记住这个口味。但是最后到了工菜三个小叛徒都没点肉夹馍。我都开始怀疑他们和老板娘的关系了。

这是肉夹馍故事的前传，后传请见回民街。

大明宫

大明宫，美其名曰四倍紫禁城之有余，据说是唐朝时的宫殿地基，有的连地基都没有了，但是丹凤门是真的好看，是我喜欢的城墙类型。这次我才不会像去北京一样傻傻地把所有的地方（颐和园）逛一遍。昔日恢弘的皇殿如今也只是饭后过来散步的平地。今天又不知在哪个景点的地图上看到了阿房宫。

戍卒叫，函谷举，楚人一炬，可怜焦土！

——杜牧《阿房宫赋》

回民街

这是一个单调地让你迷失真假的地方。室友说会不会是回族人民的街道啊，所以命名为回民街，因为里面有很多清真的东西。但是感觉和济南的芙蓉巷差远了啊，唯一能比过的怕只能是稍微宽广、干净的道路了吧。

下面说说回民街的一致性。这是我见过的最一致的小吃街（好吧，虽然我也并没有去过几条……）——

  • 招牌一致。单摊位同销售性质的招牌都几乎一致，比如都是长沙臭豆腐。这也就算了好吧，可是多摊位不同家（就是一家打着多种招牌）所打的招牌也都一样，都包括什么鱿鱼啊，红柳烤肉呀……

  • 价位一致。我也真的是醉了，自己挺喜欢吃牛肉干的，可是我们逛了纵横几条街不同商家的牛肉干的价位都是138元/斤？我已经认为他们都来自同一货源了。

  • 样式一致。同招牌的小吃的样式都一样，如果是鱿鱼也就算了，但是我在室友点鱿鱼的时候目睹旁边摊位棉花糖的制作过程……天啊，2毛不到的成本就做了个花卖了二十块钱？！而且一点技术含量都没有……于是从来没有吃过棉花糖的我高冷地迈开步伐，走向渐渐“熟悉”的另一个街头——我毫无防备地又在这里看到了同种花型的棉花糖……

  • 宣传一致。实在不明白央视记者和饭店老板多么无聊。前者脑子抽风了来这里采访随便找了几个店，然后店老板就换了一个招牌，甚至补上了视频。这会造成一个效应啊，在一致性极高的小吃街里，人们宁愿去排长队出更多的钱吃一个有着央视或者湖南卫视宣传的小吃，而周边的小吃店门可罗雀……他们竟然在这种情况下还不重换招牌？！于是我在一家央视到访过的肉夹馍小吃店外的路上被他们店里的人吆喝着排队。这没什么，央视来人到访自己底气当然要足，可是这肉夹馍让人余味无穷……

室友带我涨知识之 皮影戏

我是坚信我国的皮影戏要优于西方的人偶的。无论是牛皮剪纸、幕布光影还有舞台背后艺术家的吆喝声和乐器打击的声音，都是中华皮影艺术的体现。

之前我只是在电视上看过皮影戏。《小兵张嘎》中日军侵华后身为皮影艺术家的胖墩的爷爷被迫给日本人演出皮影戏。当然这也是他最后一次皮影戏。戏罢，胖墩爷爷用打光的油灯里的火烧掉了他所有的皮影道具——这些东西决不能落入日本人手里。

或许传统文化的魅力就是，你可能什么都听不懂，但是你能静静地坐在那里专注地观赏十来分钟。为了抢占前两排的消费区，一个室友不惜放弃购买漂亮小姐姐的剪纸而将一个红色“福”字剪纸收入囊中，在第二排开启了全程录像。至于视频能不能上传至网站上，今后有待研究。

曲罢，台下一片掌声。一个室友焦急地跑到我面前：“娘子你好漂亮啊！”

“这么精彩的皮影你就会说这一句？”

“我只听懂了这一句……”

对了，这次皮影曲目的名字是《卖货郎》。

钟鼓楼周边

没有小超市，除了KFC之类没有带饭的快餐店，甚至连面馆都没有。室友博学的经验告诉了我，这是西安一环内，如果把超市或者快餐店放一楼，后者如果没有KFC的名气，前者就更不用提了，是根本赚不了钱的。本来预计今晚夜袭西安交通大学的计划也因为觅食失败而泡汤了。

西安真的是个神奇的地方，民宿客厅窗户上写着严禁上窗外的阳台，违者罚款5000块！于是晚上十一点多当我躺在床上写博文的时候室友看到了窗外一个陌生男子在阳台徘徊的身影……

第二日

西安的交通真的很文明，尤其是“车让人”。今天我骑着凉凉的落满灰的小黄车，只是贴在人行道旁的马路上，拐角处正迎面驶来的公交车竟然减速了……后面又来了多辆小轿车，都减了速，有的竟然所幸停了车！最后羞愧的我把小黄车提到了人行道上……

东南路线

小雁塔&西安博物院

无槽点。本来想了解下小雁塔因为地震崩成两半后复原的历史和过程，但是因为太资本所以遭到我们狠心的拒绝。

省美术博物馆

这里放着很多画师的作品，有写实的，印象派和其他作品，大多数是我不看作品名称看不懂的，这之中的大部分是我看了作品名称后还是看不懂的……让你看不懂所成为的艺术也是艺术的一个小分支吧~

陕西历史博物馆

这里我室友看到了我没看到的很多东西，比如皇帝的洗澡盆……那是因为当时的我正因为困倦和劳累闭目养神，趴在专用的游客休息区里，欣赏休息区里播放的著名的钢琴曲，比如《The dawn》。

交通大学

很失望没有躺在交大的操场上看星空，等着教学楼里的某个人叫我滚蛋。只能说这些985太封闭了，和人大一样，操场都用绿网封……拯救国足，从拆下高校操场的铁网开始。

很荣幸通过了兄弟院校的钱学森图书馆身份验证，交大的图书馆有些年代了，但是里面的自习位置还是挺多的，同学也非常安静地学习，连打某联机射击游戏都很安静的呢！钱学森图书馆因为确实比较小，所以自习空间比较大，也因此没有看到过几个书架。但是图书馆的书还是比较新的（相比珞职）。

室友带我涨知识之 过桥米线

被室友南蛮头子带去了杨嫂过桥米线吃中饭。小店也有小店的特色和慷慨（云南应该也差不多，头子说云南有家规矩还多），在热情的杨嫂的指导下，我学会了过桥米线的基本吃法。期间有人（应该是西安博物院的志愿者）问杨嫂为什么不开外卖服务，杨嫂说自己现在忙成什么样了根本没时间开。

我觉得不能开，有些东西只有光临店内才能体会其魅力。

第三日

钟楼

如果这里再也不敲钟，那么叫它钟楼还有什么意思呢？除了一些旅游价值外，这里丝毫不会再有任何效用。试想一下如果还有敲钟人，每天的某个钟点可以听见一声清脆的钟响。

钟楼的卡西莫多去哪儿了？

钟鼓楼票三天内有效。

长安城墙

这是今天快乐骑行与开黑的重要环节！四人两张双人车，绕着13.7km的城墙快乐骑行一圈，其中根据自己的辛苦程度在城墙拐角处或者是瓮形城门处停车休息并进行快乐开黑。此间还对室友进行摄影教学，优秀的室友也拍摄出来了很好看的摄影作品。

室友带我涨知识之 海底捞

服务挺好的也学会了很多菜品怎么吃的方法，不说了今天要赶紧看看昨天没有看的《歌手》了。
